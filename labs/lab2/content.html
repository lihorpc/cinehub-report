Контрольні питання:

1. Для чого використовуються каскадні таблиці стилів?

Каскадні таблиці стилів (CSS) використовуються для:

  • Визначення візуального оформлення веб-сторінок
  • Розділення структури (HTML) та презентації
  • Забезпечення адаптивного дизайну
  • Створення анімацій та інтерактивних ефектів
  • Підтримки різних пристроїв та розмірів екранів
  • Спрощення підтримки та оновлення дизайну сайту

2. Які методи визначення стилів HTML-документі ви знаєте?

Існує три основні методи визначення стилів:

  • Зовнішній CSS (підключення через тег link)
  • Внутрішній CSS (використання тегу style в head)
  • Інлайновий CSS (атрибут style безпосередньо в HTML-тегах)

3. У чому різниця між принципами каскадування і спадкування?

Каскадування - це механізм визначення пріоритетності стилів при конфлікті, що враховує специфічність селекторів та порядок оголошення правил.

Спадкування - це механізм передачі властивостей від батьківського елемента до дочірнього за ієрархією DOM-дерева.

4. Як в HTML-документі оформлюються селектори тегу і клас?

Селектор тегу записується просто назвою тегу: p { }, div { }

Селектор класу починається з крапки: .class-name { }

5. Для чого використовується ідентифікатор?

Ідентифікатор використовується для:

  • Унікальної ідентифікації елемента на сторінці
  • Створення унікальних стилів для конкретного елемента
  • JavaScript маніпуляцій
  • Створення якорних посилань
  • Зв'язування елементів форм

6. Навести приклади оформлення текстових елементів

Основні властивості оформлення тексту:

  • font-family - вибір шрифту
  • font-size - розмір шрифту
  • font-weight - жирність шрифту
  • font-style - стиль шрифту
  • text-align - вирівнювання тексту
  • line-height - міжрядковий інтервал
  • letter-spacing - міжсимвольний інтервал
  • text-decoration - оформлення тексту
  • color - колір тексту

Висновки

В ході виконання лабораторної роботи №2 було досягнуто наступних результатів:

  • Освоєно різні типи CSS-селекторів (тегів, класів та ідентифікаторів) та їх практичне застосування
  • Набуто практичних навичок роботи з оформленням текстових елементів, включаючи налаштування шрифтів, кольорів та відступів
  • Вивчено принципи роботи з фоном, контурами та списками в CSS
  • Опрацьовано просунуті можливості CSS, такі як анімації, переходи, тіні та градієнти
  • Отримано досвід створення адаптивних елементів інтерфейсу (кнопки, випадаючі списки, пагінація)
  • Закріплено розуміння принципів каскадування та спадкування стилів в CSS

Отримані знання та навички є фундаментальними для подальшої роботи з веб-розробкою та створення сучасних інтерактивних веб-сторінок.
