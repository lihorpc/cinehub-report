Опис предметного середовища

Рекомендаційна система Cinehub - це інтелектуальна платформа, яка допомагає користувачам знаходити фільми та серіали на основі їхніх уподобань. Система аналізує:

  • Історію переглядів користувача
  • Оцінки та відгуки
  • Жанрові вподобання
  • Поведінку схожих користувачів

На основі цих даних алгоритм формує персоналізовані рекомендації, враховуючи популярність контенту, новинки та рейтинги. Система постійно навчається та покращує точність рекомендацій з кожною взаємодією користувача.

  • value - значення за замовчуванням

Контрольні питання

1. Яка структура HTML-документу?

HTML-документ має наступну базову структуру:


                    <!DOCTYPE html>
                    <html>
                        <head>
                            <meta charset="UTF-8">
                            <title>Заголовок сторінки</title>
                        </head>
                        <body>
                            Вміст сторінки
                        </body>
                    </html>

Де:

  • <!DOCTYPE html> - оголошення типу документа
  • <html> - кореневий елемент
  • <head> - містить метадані, заголовок, підключення стилів та скриптів
  • <body> - містить видимий контент сторінки

2. Дати визначення тега

Тег - це спеціальна конструкція мов HTML, що використовується для розмітки елементів веб-сторінки. Теги визначають структуру та семантику контенту. Більшість тегів мають відкриваючу та закриваючу частини: <тег>контент</тег>.

3. Які теги використовуються при роботі з таблицями?

  • <table> - створює таблицю
  • <tr> - створює рядок таблиці
  • <th> - створює заголовок стовпця/рядка
  • <td> - створює комірку даних
  • <thead> - групує заголовок таблиці
  • <tbody> - групує тіло таблиці
  • <tfoot> - групує підвал таблиці

4. Які теги використовуються при роботі з зображеннями?

  • <img> - вставка зображення з атрибутами:
    • src - шлях до зображення
    • alt - альтернативний текст
    • width - ширина
    • height - висота
  • <figure> - контейнер для зображення
  • <figcaption> - підпис до зображення

5. Які теги використовуються при роботі зі списками?

  • <ul> - маркований список
  • <ol> - нумерований список
  • <li> - елемент списку
  • <dl> - список визначень
  • <dt> - термін у списку визначень
  • <dd> - визначення терміну

6. Перерахувати засоби опису таблиць в HTML

  • Структурні теги: table, tr, th, td
  • Групуючі теги: thead, tbody, tfoot
  • Атрибути таблиць:
    • border - товщина рамки
    • cellpadding - внутрішній відступ в комірках
    • cellspacing - відстань між комірками
    • colspan - об'єднання стовпців
    • rowspan - об'єднання рядків
    • width - ширина таблиці
    • height - висота таблиці
