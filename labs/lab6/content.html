Створення та обробка товарів

Функція createProduct приймає два параметри:

  • obj — об'єкт товару без id
  • callback — функція зворотного виклику

Callback функції:

  • logProduct — виводить об'єкт продукту в консоль
  • logTotalPrice — обчислює і виводить загальну вартість товару

Завдання 2

Обробка медикаментів

Завдання складається з двох частин:

  1. Отримати масив назв всіх медикаментів
  2. Створити новий масив з актуальних медикаментів, відсортованих за датою

Використані методи:

  • Object.keys() — для отримання масиву назв
  • Object.entries() — для роботи з парами ключ-значення
  • filter() — для видалення прострочених медикаментів
  • sort() — для сортування за датою
  • map() — для отримання фінального масиву назв
